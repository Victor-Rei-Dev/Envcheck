
# envcheck

Verifica se programas estão instalados no sistema. Suporta verificação personalizada e tem instalador automático.

## Instalação

```bash
git clone https://github.com/seu-usuario/envcheck.git
cd envcheck
chmod +x install.sh
./install.sh
```

## Como usar e funcionalidades

```bash
# Verifica programas padrão 
envcheck

# Verifica programas específicos
envcheck docker node php

# Ajuda
envcheck --help
```
